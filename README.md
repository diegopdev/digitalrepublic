## Functional requirements:

Your development environment needs to have:

- Node JS
- Yarn

## Getting Started

```sh
# clone it
  git clone https://github.com/diegopdev/digitalrepublic.git
  cd digitalrepublic

# Install dependencies
  yarn install

# Start project development
  yarn start

# If you want to start production
  yarn build

```
